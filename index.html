LLM Training 之 DeepSpeed 参数设置

batch size 相关参数

以上三个参数选择任意一对填写即可。

train_batch_size

实际的global batch size。它的值应该严格等于train_micro_batch_size_per_gpu * gradient_accumulation_steps * gpu_numbers。
"train_batch_size": "auto|int"

train_micro_batch_size_per_gpu

每个GPU分到的batch size。
"train_micro_batch_size_per_gpu": "auto|int"

gradient_accumulation_steps

梯度累积，也即每执行多少个forward才更新一次参数。每次forward都会将当前得到的梯度与之前的梯度相加。另外该参数的增加还可以减少GPU间的通信频率。
"gradient_accumulation_steps": "auto|int"

优化器相关参数

type

包括Adam、AdamW、OneBitAdam、Lamb、OneBitLamb等 以及其他torch内部的优化器 详见(https://deepspeed.readthedocs.io/en/latest/optimizers.html#)
"type": "AdamW"

params

优化器参数，详见(https://www.deepspeed.ai/docs/config-json/#optimizer-parameters)
"params": { "lr": "auto","betas":[0.9, 0.95],"eps": 1e-8,"weight_decay": 0.01}

学习率调度相关参数

用于不断调整优化器的学习率

type

包括固定学习率、线性衰减、余弦衰减等。
"type": "WarmupLR"

params

调度器的参数
"params": {"warmup_min_lr": "auto","warmup_max_lr": "auto","warmup_num_steps": "auto"}
